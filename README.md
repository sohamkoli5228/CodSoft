# 🐍 Python Projects

This repository contains three beginner-friendly Python projects developed to strengthen programming fundamentals.

## 📌 Projects

### 📝 1. To-Do List
A command-line application to manage daily tasks.

**Features**
- Add tasks
- View tasks
- Update tasks
- Delete tasks
- Save tasks using file handling

---

### 🧮 2. Calculator
A simple calculator that performs basic arithmetic operations.

**Features**
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero

---

### 🔐 3. Password Generator
A Python program that generates strong and random passwords.

**Features**
- User-defined password length
- Random password generation
- Letters, numbers, and symbols

---

## 🛠️ Technologies Used

- Python 3
- File Handling
- Functions
- Loops
- Conditional Statements
- `random` and `string` modules

## ▶️ How to Run

```bash
python filename.py
```

Replace `filename.py` with:
- `todo.py`
- `calculator.py`
- `password_generator.py`
